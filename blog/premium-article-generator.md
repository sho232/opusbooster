# 高品質記事生成AIの使い方

## 概要

高品質記事生成AIは、テーマとキーワードを入力するだけで、SEOに配慮したブログ記事を5つのステップで作成できる機能です。タイトル案の提案、記事構成(見出し)の生成と編集、本文の自動執筆、目次・FAQの自動挿入までを、ひとつの画面で完結できます。

「とりあえず生成」ではなく、**タイトルと構成を自分で確認・調整してから本文を書かせる**流れになっているため、狙いに合った記事に仕上がりやすいのが特長です。

## 起動方法

ブログ記事の編集画面を開くと、タイトル入力欄の上に「**✨ 高品質記事を作成**」ボタンが表示されます。クリックすると、生成ウィザードが開きます。

## ステップ1: 基本情報

まず、記事の土台になる4つの項目を入力します(すべて必須)。

| 項目 | 入力例 |
|---|---|
| 記事のテーマ・トピック | 初心者向けヨガの始め方 |
| 主要キーワード | ヨガ 初心者 始め方 効果 自宅 |
| ターゲットオーディエンス | 20〜40代の運動不足の女性、健康に関心がある会社員 |
| 記事の意図・目的 | ヨガの基礎知識を提供し、自宅でも始められることを伝える |

入力できたら「**タイトル候補を生成**」をクリックします。

※画面上部のURL自動入力欄は現在準備中の機能です。

## ステップ2: タイトル選択

入力した情報をもとに、SEOを考慮した45〜60文字程度のタイトル案が10個提示されます。

- 気に入ったタイトルをクリックして選択します
- しっくりくるものがなければ「**タイトル再生成**」で作り直せます
- 選択したら「**選択したタイトルで続行**」をクリックします

## ステップ3: 記事構成

選んだタイトルに基づいて、記事の見出し構成(導入 → 本文の見出し5〜7個 → まとめ)が自動生成されます。この構成はそのまま使うこともできますが、自由に編集できます。

- **見出しの文言を変える**: 見出しテキストをクリックして直接編集
- **見出しレベルを変える**: 「H2 / H3」ボタンをクリックして大見出し⇔小見出しを切り替え
- **見出しを消す**: 「×」ボタンで削除
- **見出しを足す**: 「+ 見出しを追加」で新しい見出しを追加
- **最初からやり直す**: 「構成再生成」で構成全体を作り直し

構成が固まったら「**この構成で続行**」をクリックします。

## ステップ4: 詳細設定

記事の仕上がりを細かく調整します。すべて任意項目です。

- **目標文字数** — 未入力の場合は2,000〜3,000文字程度で生成されます
- **文体** — フォーマル(ですます調) / カジュアル(だである調) / 専門的 / シンプル(初心者向け) / ストーリー調 の5種類
- **トーン** — ニュートラル / 熱意あふれる / 情報提供型 / 説得力のある / 共感する の5種類
- **独自のポイント・主張** — 記事に盛り込みたいオリジナルの視点や体験を箇条書きで
- **紹介したい商品・サービス** — 記事内で自然な形で紹介したい自社の商品・サービス
- **コールトゥアクション** — 記事の最後に入れたい行動の呼びかけ(例: 無料体験レッスンに申し込む)

**高度なSEO設定**では、以下のオン/オフを選べます(初期状態はすべてオン)。

- FAQ(よくある質問)セクションを追加
- 目次を自動生成
- メタタイトル・ディスクリプション提案

設定できたら「**記事を生成**」をクリックします。

## ステップ5: 記事生成

進捗バーが表示され、「記事構造の構築 → 各セクションの本文生成 → 最終化」の順で記事が作られます。内容によって数十秒〜数分かかりますので、そのままお待ちください。

生成が完了すると、記事のプレビューが表示され、次の操作ができます。

- **記事を再生成** — 同じ設定でもう一度生成し直します
- **記事をコピー** — 記事のHTMLをクリップボードにコピーします
- **エディタに挿入** — タイトルはブログのタイトル欄に、本文は編集中の記事本文に自動で挿入されます

## 生成される記事に含まれるもの

- **目次** — 各見出しへのジャンプリンク付き(オンにした場合)
- **構造化された本文** — 大見出し(H2)・小見出し(H3)で整理された読みやすい構成
- **FAQセクション** — 記事内容に関連した「よくある質問」5問(オンにした場合)
- **コールトゥアクション** — 設定した行動の呼びかけを記事末尾に配置

## 活用のヒント

- **キーワードは3〜5個程度**に絞ると、焦点の定まった記事になります
- **「独自のポイント・主張」にご自身の体験や実績を入れる**と、他にはないオリジナリティのある記事になります
- ターゲットオーディエンスは「30代女性」のような属性だけでなく、「運動不足に悩んでいる」「時間がない」といった**悩みや状況まで書く**と、読者に刺さる内容になりやすいです
- 生成された記事は公開前に必ずご自身で内容を確認し、事実関係や数値に誤りがないかチェックしてください
