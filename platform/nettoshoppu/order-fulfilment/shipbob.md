# ShipBob連携

ShipBob連携を使うと、管理の主導権を手放すことなくフルフィルメントを外部に委託できます。ネットショップをShipBobのグローバルなフルフィルメントネットワークに直接接続することで、在庫の保管、注文フルフィルメントの自動化、顧客へのより迅速な配送が可能になります。

ストアで受けた注文は、ピッキング・梱包・発送のために自動的にShipBobへ送信されるため、手作業は必要ありません。分散型の倉庫とリアルタイム同期により、事業の拡大を見据えたブランドに適した連携です。

## 接続手順

1. ShipBobのダッシュボードでAPIトークンを作成します
2. ネットショップのタブに戻ります
3. 「**設定**」を選択します
4. 「**連携を追加**」を選択します
5. 「**ShipBob**」を選択します
6. 表示されたポップアップに、ShipBobのAPIキーを貼り付けます

これで、ネットショップとShipBobの連携は完了です。
