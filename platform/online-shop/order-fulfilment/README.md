# 注文フルフィルメント

ネットショップの注文フルフィルメント（受注後の出荷・配送業務）は、外部のフルフィルメントサービスと連携することで自動化できます。

OpusBoosterでは、以下の4つのサービスとの連携に対応しています。

* **ShipStation** — 配送ラベルの作成やキャリア管理を一元化できる出荷管理サービス
* **Shippo** — 複数キャリアの料金比較とラベル印刷ができる配送プラットフォーム
* **ShipBob** — 在庫保管から出荷までを代行するフルフィルメントサービス
* **Printful** — プリントオンデマンド商品の生産・発送を代行するサービス

各サービスの接続手順は、それぞれのページで詳しく説明しています。
