# パスワード再設定

他のメールテンプレートと同様に、パスワード再設定メールもドラッグ＆ドロップエディターで自由にカスタマイズできます。

### テンプレートにあらかじめ設定されているもの

* **システムフィールド** — ウェブサイト名と、再設定に必要な情報フィールドが割り当て済みです。
* **既定のコンテンツ** — ユーザーがパスワードを再設定するための手順があらかじめ設定されています。

### カスタマイズ方法

* **フィールドの変更・削除** — システムのプレースホルダーは必要に応じて調整できます。
* **ドラッグ＆ドロップエディターを使う** — レイアウト、カラー、ブランド要素をかんたんにパーソナライズできます。
* **文面の強化** — わかりやすく親切な文章にして、迷わずパスワードを再設定できるようにしましょう。

安全でブランドらしいパスワード再設定の体験を提供するための、シンプルながら欠かせない設定です。

### フィールドを追加するには

システムメールのテンプレートにフィールドを追加したい場合は、テキスト入力中にテキストエディターを選択し、**タグ**アイコンをクリックします。タグアイコンをクリックすると、そのシステムテンプレートに追加できる専用フィールドが一覧表示されます。

ここには、パスワード再設定のシステムメールに割り当てられたすべての専用フィールドが表示されます。

また、すべてのCRMプロパティもメールに追加できます。自分で作成したカスタムプロパティがある場合は、それらもここに一覧表示されます。
