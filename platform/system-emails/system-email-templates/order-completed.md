# 注文完了

注文完了メールも、ドラッグ＆ドロップのメールエディターでカスタマイズして、洗練されたプロフェッショナルなコミュニケーションを保てます。

### テンプレートにあらかじめ設定されているもの

* **システムフィールド** — ウェブサイト名、注文のサマリー、割り当て済みのフィールド。
* **既定のコンテンツ** — 完了の確認、次のステップの案内、お礼のメッセージがあらかじめ設定されています。

### カスタマイズ方法

* **フィールドの変更・削除** — プレースホルダーはブランドに合わせて調整できます。
* **ドラッグ＆ドロップエディターを使う** — レイアウト、カラー、構成をかんたんにパーソナライズできます。
* **文面の見直し** — わかりやすく心のこもった内容にして、気持ちの良い購入体験で締めくくりましょう。

お客様に安心と満足を届けながらブランドの印象を高める、手軽な方法です。

### フィールドを追加するには

システムメールのテンプレートにフィールドを追加したい場合は、テキスト入力中にテキストエディターを選択し、**タグ**アイコンをクリックします。タグアイコンをクリックすると、そのシステムテンプレートに追加できる専用フィールドが一覧表示されます。

ここには、注文完了のシステムメールに割り当てられたすべての専用フィールドが表示されます。

また、すべてのCRMプロパティもメールに追加できます。自分で作成したカスタムプロパティがある場合は、それらもここに一覧表示されます。
