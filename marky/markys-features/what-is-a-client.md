# クライアントとは?

## 代理店のクライアントとは?

「クライアント」とは、代理店(エージェンシー)アカウントでのみ利用できる特別なユーザー種別です。クライアントの特徴については、以下の表をご覧ください。

| 項目 | 代理店のクライアント |
|--------|----------------|
| **利用シーン** | クライアントが自社のSNS運用を代理店に依頼している場合 |
| **ログイン方法** | 専用URL・専用ポータル経由 |
| **UIのブランディング** | 代理店独自のブランディング(Markyの痕跡は一切表示されません) |
| **アプリ内課金** | クライアントにはアプリ内課金やペイウォールが一切表示されません |
| **アプリ内アンケート** | クライアントに利用体験に関するアンケートは送られません |
| **広告ターゲティング** | Markyはこのユーザーを広告パートナーにリターゲティング対象として報告しません |
| **事業管理** | 代理店が管理します |
| **サポート・トレーニング** | 代理店が提供します |
| **メール配信** | Markyからクライアントへメール(マーケティング、育成、教育系など)は一切送られません |
| **権限設定** | 代理店が細かく権限を設定可能(アドバンスドホワイトラベルの機能) |
| **データプライバシー** | 代理店とMarkyの両方のプライバシーポリシーに準拠します |
| **通知** | Markyからの通知メールは届きません(パスワード再設定・投稿失敗の通知なども含む) |

## クライアントの追加方法

1. まず、代理店アカウントの「クライアントポータル」設定を確認してください。

![domain](https://storage.crisp.chat/users/helpdesk/website/1de1cb83edbcfb00/image_1i5rj75.png)

2. クライアントを追加したいビジネスに移動し、「クライアントを追加」をクリックします

![Add Client](https://storage.crisp.chat/users/helpdesk/website/1de1cb83edbcfb00/image_18rsamp.png)

3. Markyが発行するリンクをクライアントに送り、アカウントを設定してもらいます

![link](https://storage.crisp.chat/users/helpdesk/website/1de1cb83edbcfb00/image_1so8cn3.png)

4. クライアントはこのリンクを使って、あなたのホワイトラベルドメイン上でアカウントを設定します

![Setup](https://storage.crisp.chat/users/helpdesk/website/1de1cb83edbcfb00/image_pzx7zl.png)

5. これで完了です！クライアントはいつでもあなたのクライアント用ドメインにログインし、投稿を確認できるようになります。

![Client Portal](https://storage.crisp.chat/users/helpdesk/website/1de1cb83edbcfb00/image_1s6qc81.png)

**豆知識:** クライアントは、クライアントポータルの「ビジネス」タブから自分のSNSアカウントを連携させることもできます。

利用可能なホワイトラベル機能について詳しく知りたい方は「[スタンダードホワイトラベル vs アドバンスドホワイトラベル](standard-vs-advanced-white-labeling.md)」をご覧ください。

レビュープロセスの仕組みについて詳しく知りたい方は「[代理店レビュープロセス: ステータスとコメント](review-process-statuses-and-commenting.md)」をご覧ください。
