# Markyについて

MarkyはOpusBoosterのSNS自動投稿AIの基盤システムです。投稿デザインの作成からソーシャルアカウントの接続、レビュー、公開まで、SNS運用に必要な一連の作業をひとつのワークスペースで完結できます。

このセクションでは、Markyのはじめ方から機能の使い方、SNS運用の実践的なノウハウまでをまとめています。
