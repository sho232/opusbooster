# コミュニティウィジェット

コミュニティウィジェットは他のウィジェットと同様に機能します — あなたは **ドラッグアンドドロップ** して **任意のページ、ファネル、ブログ** に入れることができ、そしてはい、あなたの **コミュニティサークル** 内でも簡単にアクセスして関与できるようになります。

それらは同じ **ウィジェットエリア** に配置されており、レイアウト内で簡単に見つけて統合できるようになっています。これにより滑らかで直感的な体験が保証され、コミュニティのニーズに合わせたインタラクティブな機能でサークルを強化できます。
