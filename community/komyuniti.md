# コミュニティ通知

メンバーを維持し続けること — 情報提供・エンゲージメント・つながりを保つことは、リテンションと交流を高める鍵です。コミュニティ通知設定を使えば、次のことができます：

* メンションでメンバーに通知 – ユーザーを直接タグ付けして注意を引きます。
* システムメールを送信 – コミュニティ通知を見逃しても、メンバーに通知が届くようにします。
* 高いエンゲージメントを維持 – 自動更新により、コミュニティが常に活発で反応的な状態を保てます。

シンプルで効果的な方法でコミュニケーションを強化し、メンバーの参加を促進！

{% hint style="success" %}
プロのヒント：通知は関連性とバランスが肝心

通知はメンバーの関与を維持するのに有効ですが、使い過ぎは侵入的に感じられることがあり、離脱につながることもあります。

* 節度を持って使う — 更新は本当に価値がある、または時間に敏感なものだけに。
* スパム化を避ける — 通知が多すぎると、メンバーは無視し始めるかもしれません。
* パーソナライズされたメンション — ユーザーをタグ付けするのは、必要な場合に限り、メッセージが意味のあるものとして伝わるようにしましょう。
{% endhint %}

### コミュニティ通知を有効にする

有効化するには、通知をオンにしてメンバーの関与を維持しましょう。

{% stepper %}
{% step %}
### 設定メニューを開く

* 左サイドバーへ移動して設定の歯車アイコンを見つけます。
* 「通知設定」をクリックすると、利用可能なすべての通知オプションが表示されます。

![通知メイン設定](<../.gitbook/assets/image (4)>)
{% endstep %}

{% step %}
### 主要オプションを確認する

設定メニューには、次の2つの重要なオプションがあります：

* 通知を有効化 – これをオンにすると、メンバーは更新、メンション、システム通知を受け取れます。
* ダークテーマを使用 – ダークなインターフェースに切り替えると、目に優しい体験になります。
{% endstep %}

{% step %}
### ビルダーキャンバスでの通知タブ

通知を有効にした後、設定メニューに戻ってビルダーキャンバスを確認します。

* 通知タブは画面右上隅にあります。
* このタブでアラート、メンション、システム更新を表示でき、リアルタイムで情報を把握できます。

![通知ベル](<../.gitbook/assets/image (5)>)
{% endstep %}
{% endstepper %}

### 通知サイドバー

ユーザーが通知アイコンをクリックすると、スライドアウトボックスが表示され、関連アップデートが一覧されます：

* メンションとアラート – やり取りやタグ付けされたメッセージを直接ハイライト。
* コメントと投稿 – ユーザーがすばやく会話へジャンプでき、通知の発生元へ移動します。
* シームレスなナビゲーション – アラートをクリックすると、ユーザーは関連する投稿・コメント・メンションに直接移動し、即時アクセスが可能です。

![通知スライドアウト](<../.gitbook/assets/image (6)>)

### システムメール

#### 未読通知の自動メールリマインダー

メンバーが反応しない場合、一定時間内（20分）に通知が未読のままだと、システムは自動で「コミュニティメンション」メールを送信します。

仕組み：

* 重要な更新を見逃さないようにメール通知を届けます。
* ユーザーがプラットフォームから離れていても、アクティブな関与の維持に役立ちます。
* 受信者はメールから関連する投稿・コメント・メンションに直接移動でき、シームレスにアクセスできます。

[コミュニティメンション](https://buildersupport.gitbook.io/help-docs/ja/purattofmu/shisutemumru/shisutemumrutenpurto/komyunitimenshon)

### メンションの使い方

メンションは、メンバーの注意を引く、またはグループ全体に一度に通知する強力な方法です。

* 個々のメンバーをタグ付け – @username で誰かを直接メンションし、通知を確実に届けます。
* 全員に通知 – @everyone でコミュニティ内の全メンバーにアラートを送り、重要な更新に最適です。
* エンゲージメントを高める – メンションは迅速な返信を促し、会話を活発に保ちます。

使い方：活用すればメンバーを圧倒せずに交流を維持しましょう！

{% hint style="success" %}
注意：全員をメンションするには、@all と @everybody も使用できますが、これらのタグを使うにはコミュニティの管理者（ADMIN）である必要があります。
{% endhint %}
