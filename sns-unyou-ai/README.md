# SNS運用AIについて

SNS運用AIは、OpusBoosterのSNS自動投稿AI機能です。投稿デザインの作成からソーシャルアカウントの接続、レビュー、公開まで、SNS運用に必要な一連の作業をひとつのワークスペースで完結できます。

このセクションでは、SNS運用AIのはじめ方から機能の使い方、SNS運用の実践的なノウハウまでをまとめています。
