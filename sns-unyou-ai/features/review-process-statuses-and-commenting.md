# レビュープロセス: ステータスとコメント

## 代理店とクライアントのレビュープロセス

このドキュメントでは、代理店とクライアントが効果的に連携するためのレビューワークフローについて説明します。クライアントの概要や設定手順については「[代理店のクライアントとは?](what-is-a-client.md)」もあわせてご覧ください。

### 仕組み

* 代理店は、任意の投稿を「レビュー待ち」としてマークできます
* これらの投稿はクライアントのダッシュボードに表示されます
* クライアントは投稿を「承認」または「修正依頼」でき、コメントを残すこともできます
* 代理店はそのコメントに対応し、承認された投稿を公開できます

## ステップバイステップ

### 代理店側の操作

1. 「生成済み」タブまたは「下書き」タブで、投稿の内容に問題がなければ「レビュー待ちにする」をクリックし、投稿のステータスを更新します。

![](https://storage.crisp.chat/users/helpdesk/website/1de1cb83edbcfb00/image_1bnde1g.png)

2. 投稿には「レビュー待ち」のステータスが付きます。この投稿はクライアントのダッシュボードに表示され、クライアントがレビューできる状態になります。

現時点ではクライアントへの通知は手動で行っていただく必要がありますが、今後通知機能を追加予定です。

![](https://storage.crisp.chat/users/helpdesk/website/1de1cb83edbcfb00/image_162azni.png)

3. 投稿にコメントを追加して、補足情報を伝えることもできます。

![](https://storage.crisp.chat/users/helpdesk/website/1de1cb83edbcfb00/image_1f0nnex.png)

4. クライアントがステータスを更新すると、「修正依頼」または「承認済み」に変わります。ここから必要な修正を行うか、そのままスケジュール・公開に進めます。

### クライアント側の操作

1. クライアントにクライアントポータルからログインしてもらいます。設定方法は「[代理店のクライアントとは?](what-is-a-client.md)」をご覧ください。

2. ログイン後、クライアントは代理店が「レビュー待ち」にした投稿をすべて確認できます。投稿の承認、修正依頼、コメントの追加が可能です。

![](https://storage.crisp.chat/users/helpdesk/website/1de1cb83edbcfb00/image_138eyy1.png)
