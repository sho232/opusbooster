# スタンダードホワイトラベル vs アドバンスドホワイトラベル

## スタンダードとアドバンスド、何が違う?

代理店アカウントでは、すべてのビジネスに標準でスタンダードホワイトラベルが付属しています。アドバンスドホワイトラベルは、任意のビジネスに追加料金でオプション追加できます。両者の違いは以下の表をご覧ください。

| 機能 | スタンダードホワイトラベル | アドバンスドホワイトラベル |
|---------|------------------------|-------------------------|
| ホワイトラベル化されたクライアントポータル | ✅ | ✅ |
| 承認プロセス | ✅ | ✅ |
| コメント機能 | ✅ | ✅ |
| SNS連携 | ✅ | ✅ |
| 新規投稿の生成 | ❌ | ✅ |
| 投稿の編集 | ❌ | ✅ |
| ビジネス詳細の編集 | ❌ | ✅ |
| カスタム権限設定 | ❌ | ✅ |

## ユーザーごとのカスタム権限はどう設定する?

まず、カスタム権限を設定するには、そのビジネスがアドバンスドホワイトラベルを利用している必要があります。設定できれば、そのビジネス内の任意のクライアントについて権限を変更できます。ビジネス内のクライアントページに移動し、クライアントの権限タイプの隣にある編集ボタンをクリックすると、そのクライアントに対して有効・無効にしたい権限を選択できます。

## ビジネスのホワイトラベルをアップグレード/ダウングレードするには?

代理店ダッシュボードで、ドロップダウンから希望するホワイトラベルの種類を選択してください。

## アドバンスドホワイトラベルの契約数を確認するには?

アカウントページで、アドバンスドプランの契約数を含む料金の内訳を確認できます。ビジネスのアップグレード/ダウングレードは代理店ダッシュボードから行えます。

クライアントについてさらに詳しく知りたい方は「[代理店のクライアントとは?](what-is-a-client.md)」をご覧ください。
