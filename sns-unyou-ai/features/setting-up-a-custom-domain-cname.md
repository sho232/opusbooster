# カスタムドメイン(CNAME)の設定方法

## CNAMEとは?

代理店は、それぞれ独自のクライアントログインポータルを運用できます。CNAME(カスタムネーム)を設定すると、標準ドメインを、`social.yourdomain.com` のようなご自身のドメインに置き換えることができます。これにより、投稿の配信やクライアントとのやり取りにおいて、より専門的な印象を与え、一貫したブランディングを維持できます。

## CNAMEを使うメリット

- **プロフェッショナルなブランディング**: 自社ドメインを使うことで、クライアントに対して専門的な印象を与えられます。
- **一貫性**: すべての投稿やリンクが自社ブランドのドメインから発信されるため、ブランドの一貫性が強化されます。
- **信頼性の向上**: ユーザーは見慣れた、信頼できるドメインの方が安心して利用しやすくなります。

## CNAME設定の手順

### 1. ドメインを用意する

- **すでにドメインをお持ちの場合**: 素晴らしいです。このあとの手順でそのまま使用します。
- **まだドメインがない場合**: Namecheap、GoDaddyなど、同様のDNSプロバイダーから取得できます。

### 2. カスタムドメインを決める

- ドメインをお持ちの場合は、`social.yourdomain.com` のように、ブランドに合わせたサブドメインを指定してください。
- **または**、弊社が提供する**シンプルドメイン**オプション(`yourprefix.socialmediapostmanager.com`)を選択することもできます。

### 3. DNSにCNAMEレコードを追加する

ドメインの所有権を確認し、サーバーに向けて設定するため、DNSに2つのレコードを追加していただく必要があります。

- DNSプロバイダーの管理画面にログインします。
- CNAMEレコードを追加します。
  - **Name/Host**: カスタムのプレフィックスを入力します(例: `social.yourdomain.com` の場合は「social」)。
  - **Type**: **CNAME** に設定します。
  - **Value/Points To**: SNS運用AIが指定するCNAMEの接続先を入力します。
- **レコードを保存**し、反映されるまで数分お待ちください。

### 4. 設定側で確認する

- ステータスが「完了」と表示されたら、新しいウィンドウでポータルを開いてみてください。ログイン画面が表示されれば、CNAMEの設定は成功です。

**補足**: クライアントポータルは、クライアント専用(代理店オーナーは対象外)のログイン画面です。クライアント以外のメールアドレスでログインを試みても、ログインすることはできません。
