# 🔧 よくあるつまずきと解決方法

設定がうまくいかないときは、このページのチェックリストを**上から順に**確認してください。多くのつまずきは、ここにある確認だけで解決します。解決しない場合の問い合わせ方法は、ページの最後にまとめてあります。

## 📩 メールが届かない <a href="#email" id="email"></a>

「テスト送信したのに届かない」「登録者にメールが届いていないと言われた」というときの確認手順です。

1. **迷惑メールフォルダを確認する** — まず受信側の迷惑メール（スパム）フォルダとプロモーションタブを見てください。ここにある場合は、次の2番の設定で改善します
2. **メールドメインの接続を確認する** — [メールドメインの接続](../platform/email-and-automation/email-domain-connection.md)の設定（DNSレコードの登録）が完了しているか、設定画面で認証済みになっているかを確認します。この設定がないと、迷惑メール扱いされやすくなります
3. **メールアカウント認証を確認する** — [メールアカウント認証](../platform/email-and-automation/email-account-verification.md)が済んでいるかを確認します
4. **宛先リストの状態を確認する** — 相手が配信停止（購読解除）していないか、[メールリスト](../platform/email-and-automation/email-lists.md)で確認します。ダブルオプトインを使っている場合は、[確認メール](../platform/system-emails/system-email-templates/double-opt-in-confirmation.md)を相手がクリックし終えているかも確認してください
5. **送信テストをやり直す** — 設定を変えたら、[個別のメール送信](../platform/email-and-automation/individual-emails.md)で自分宛てにテストし、受信トレイに入るかを確認します

## 🌐 ドメインが反映されない <a href="#domain" id="domain"></a>

「カスタムドメインを設定したのにサイトが表示されない」というときの確認手順です。

1. **まず待つ** — DNSの変更は世界中に行き渡るまで**数時間〜最大48時間**かかります。設定直後に表示されないのは異常ではありません
2. **DNSレコードの値を見直す** — [カスタムドメインの接続](../platform/getting-started/custom-domain.md)に記載の値と、お使いのドメイン会社（お名前.com、ムームードメインなど）の管理画面に入力した値が**完全に一致**しているか確認します。余分なスペースや、コピー漏れがよくある原因です
3. **別の回線・端末で開いてみる** — お使いの回線やブラウザに古い情報が残っていることがあります。スマホの4G/5G回線で開く、ブラウザのシークレットモードで開く、を試してください
4. **鍵マーク（SSL）のエラーの場合** — ドメイン接続後、暗号化証明書の発行に少し時間がかかることがあります。1〜2時間おいてから再度アクセスしてください

## 💳 決済がつながらない <a href="#payment" id="payment"></a>

「購入ボタンを押してもエラーになる」「テスト購入ができない」というときの確認手順です。

1. **決済サービスの接続状態を確認する** — [決済サービスとの接続](../platform/online-shop/payment-services.md)で、決済サービス（Stripeなど）との連携が完了しているかを確認します。決済サービス側でアカウントの本人確認が終わっていないと、決済が通らないことがあります
2. **商品の価格・通貨設定を確認する** — [商品の作成](../platform/online-shop/create-products.md)で、価格と通貨（JPY）が正しく設定されているかを確認します
3. **テストモードと本番モードを確認する** — 決済サービス側がテストモードのままだと、実際のカードでは決済できません。逆に本番モードでは、テスト用カード番号は使えません
4. **注文メールで動作確認する** — 決済が通ったら、[新規注文メール](../platform/system-emails/system-email-templates/new-order.md)が届くかも合わせて確認しておくと安心です

## 🔄 その他の「うまく動かない」への共通対処

どの機能でも、まずこの3つを試すと解決することが多いです。

* **ページを再読み込みする** — 特に設定を変えた直後は、Ctrl+Shift+R（Macは Cmd+Shift+R）の強制再読み込みが有効です
* **シークレットモードや別のブラウザで試す** — ブラウザの拡張機能やキャッシュが干渉していないかを切り分けられます
* **少し時間をおく** — DNS・SSL・メール認証など、外部との連携を伴う設定は反映に時間がかかるものがあります

## 🙋 それでも解決しないときは

[サポート](getting-support.md)からお問い合わせください。その際、次の3点を添えていただくと、解決までがぐっと速くなります。

1. **問題が起きているページのURL**
2. **エラー画面のスクリーンショット**
3. **何をしようとして、どう操作したか**（例: 「商品Aの購入ボタンを押したら〜というエラーが出た」）

自分で設定を進めるのが難しいと感じたら、Zoomでのサポートや構築の代行についてもお気軽にご相談ください。
