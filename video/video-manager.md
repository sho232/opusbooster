# ビデオマネージャー

### ネイティブ動画ホスティングとは

外部の動画サービスに頼らず、プラットフォーム内で動画をアップロード・管理・設定できる機能です。ブランド体験を保ちながら、動画配信をスムーズに行えます。

### アップロード後にできること

* 設定を細かくコントロール：再生オプション、公開範囲、エンゲージメント機能などを調整できます。
* 表示方法を柔軟に選択：埋め込み、レイアウト調整、見え方の指定が可能です。
* パフォーマンス最適化：外部プラットフォームの制限に左右されにくく、意図した形で提供できます。
* 他機能と連携：コミュニティ投稿、ディスカッション、学習モジュールなどに自然に紐づけられます。

### 動画のアップロード手順

1. 左側のタスクバーから、サイト設定の「歯車」をクリック
2. 上部にある「ビデオ」を選択
3. ポップアップが開いたら、目的に応じて操作します

![](../.gitbook/assets/videomanager.png)

ポップアップが表示されると、ビデオの整理と管理に関していくつかのオプションが利用できます：

* **新しいビデオを追加** – 新しいコンテンツをプラットフォームに直接アップロードします。
* **フォルダを作成** – カスタムフォルダにグループ化してビデオを整理します。
* **既存ビデオを編集** – 以前アップロードしたコンテンツの設定や表示設定を変更します。

![](../.gitbook/assets/videomanagerpopup.png)

{% hint style="success" %}
**便利な整理のコツ（コースやプレイリストを作る場合）**\
コースや連続コンテンツ（プレイリスト）を作る予定があるなら、最初にフォルダを分けておくのがおすすめです。

**フォルダ分けのメリット**\
・管理が楽：目的のレッスン動画をすぐ見つけて更新できます\
・探しやすい：視聴者側も内容を追いやすくなります\
・拡張しやすい：コンテンツが増えても構造が崩れにくいです
{% endhint %}

例：作成した「My Videos」フォルダを開き、表示される「+Upload A Video」からアップロードを開始します。

![](<../.gitbook/assets/image (2)>)

ボタンを押すとPC内のファイル選択画面が開くので、動画を選ぶと「Video settings」画面に進みます。

![](<../.gitbook/assets/image (3)>)

### **ビデオタイトル**

分かりやすいタイトルを付けると、プラットフォーム全体で扱いやすくなります。たとえば以下の場面で表示・参照されます。\
・プレイリストに追加するとき（識別しやすい）\
・コースのカリキュラムに組み込むとき（受講者が迷いにくい）\
・ライブラリで管理するとき（後から探しやすい）

**ビデオ説明**

内容の補足や前提、見どころ、重要ポイントなどを書けます。プレイリストやコース等に追加したときにも表示されます。

説明文を入れるメリット

* 動画の目的が伝わる
* 補足情報や目印（タイムスタンプ等）を書ける
* プラットフォーム内検索で見つけやすくなる可能性がある

{% hint style="success" %}
**表示をスッキリさせたい場合**\
ページへ埋め込む際に、タイトルや説明文の表示をオフにできる場合があります。余計なテキストを減らし、デザインに馴染ませたいときに便利です。
{% endhint %}

**サムネイル**

ビデオサムネイルは再生前にビデオを表すプレビュー画像です。カスタムサムネイルをアップロードしてビデオを視覚的に魅力的で認識しやすくできます。

効果：

* より良いエンゲージメント – 魅力的なサムネイルは視聴数を増やす可能性があります。
* 明確な識別 – ユーザーがビデオのトピックをすばやく認識できるようにします。
* ブランドの一貫性 – ビジュアルを全体のデザインと整合させます。

サムネイルを追加すると、組み込みのファイルマネージャーで直接開きます。

[ファイルマネージャーの使い方](https://buildersupport.gitbook.io/help-docs/ja/purattofmu/webusaitobirud/fairumanjnoi)

**カテゴリ**

ビデオをカテゴリに整理することは、特にビデオプレイリストウィジェットのフィルタリングタブを使用する場合に、ユーザーが特定のコンテンツを見つけやすくします。

カテゴリ分けの利点：

* ナビゲーションの改善 – カテゴリで絞り込んで関連ビデオを素早く見つけられます。
* コンテンツの整理維持 – 異なるタイプのビデオを明確に構造化するのに役立ちます。
* ユーザーエクスペリエンスの向上 – 閲覧プロセスを簡素化し、発見を容易にします。

![](<../.gitbook/assets/image (4)>)

**ビデオチャプター**

ビデオチャプターにより、視聴体験を向上させるためにビデオを複数のセクションに分割できます。

主な機能：

* 複数のチャプター – コンテンツを構造化されたセグメントに分割して整理を向上させます。
* ツールチップのポップアウト表示 – チャプターにカーソルを合わせると、選択前にその詳細をプレビューします。
* 効率的な再生 – 最も重要なセクションに直接ジャンプできます。

ビデオチャプターにより、ユーザーは重要な瞬間を素早く見つけることができ、長尺コンテンツをより魅力的でアクセスしやすく、インタラクティブにします。

ビデオをチャプターに整理すると、視聴者が重要な瞬間を効率的にナビゲートしやすくなります。手順は次のとおりです：

* 「Video Chapters」タブを選択して、チャプター管理ウィンドウにアクセスします。
{% endstep %}

{% step %}
* 「Add Chapter」をクリックして、新しいチャプターの作成を開始します。
{% endstep %}

{% step %}
* タイムスタンプを選択 — チャプターをビデオ内の参照したい正確な位置に配置します。

* チャプターを編集 – チャプター名を変更したり位置を調整してナビゲーションを洗練できます。
* チャプターを削除 – 不要なチャプターを削除してビデオを整理します。

**キャプション**

キャプションを追加することで、ビデオのアクセシビリティが向上し、視聴者の理解が深まります。アップロードと管理の方法：

* ビデオ設定で「Captions」オプションをクリックします。
* キャプションファイルをアップロードします（VTT形式を使用）。
* 言語を選択し、自動表示（オン/オフ）を設定できます。
* 再生時にキャプションは表示され、視聴者はビデオプレーヤーの CC アイコンをクリックして有効/無効を切り替えられます。

この機能はアクセシビリティを向上させ、視聴者が体験を制御できるようにします。

![](<../.gitbook/assets/image (9)>)

**埋め込みと共有**

「Share」をクリックして、埋め込みオプションにアクセスします。設定を構成して、外部プラットフォーム、メール、またはビデオ埋め込みをサポートするあらゆる場所での表示設定（可視性、自動再生、カスタム表示設定など）を調整します。埋め込みスクリプトをコピーして、ウェブサイト、メール、または埋め込みを許可する他のプラットフォームに貼り付けます。

仕組み：

* シームレスなコンテンツ統合により、ビデオをネイティブプラットフォームの外でもアクセス可能にします。
* 埋め込む際に構成した追加オプション（例：キャプションやチャプター）も含まれます。
* これにより、異なるプラットフォーム間で一貫した表示が確保され、インタラクティブ要素が維持されます。

フルカスタマイズでビデオを埋め込むことで、視聴者はどこで共有されても構造化ナビゲーション、アクセシビリティ機能、再生設定の恩恵を受け続けます。
