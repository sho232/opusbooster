# Contributing to OpusBooster Docs

Thanks for your interest in contributing!  
We welcome fixes, improvements and new examples.

## How to contribute

1. **Fork** this repository and create a new branch:  
   `git checkout -b my-fix`
2. Make your changes following our style guide (see below).
3. **Commit** the changes and open a **Pull Request**.
4. A maintainer will review your PR and merge it once approved.

## Style guide

- Write in clear, concise Japanese or English.
- Use Markdown headings (`##`) to structure sections.
- For code blocks, specify the language (` ```js `, ` ```bash `).

## Reporting issues

Open a GitHub Issue and describe:
- What page/section is affected
- Expected behavior vs. actual behavior
- Steps to reproduce (if applicable)

Thank you for helping us improve the docs!
