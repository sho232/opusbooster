# 🗺️ 目的別サクセスマップ

このページは、「**やりたいこと**」から逆算して、必要な設定と手順を順番にたどれる道案内です。機能の名前がわからなくても大丈夫です。あなたのゴールに近い項目から始めてください。

## まず、あなたはどのフェーズ？

* **これからビジネスを始める** → [共通セットアップ](success-map.md#setup)を済ませてから、[発信の仕組み](success-map.md#content)か[メールリスト集め](success-map.md#leads)へ
* **売る商品・サービスはある** → [共通セットアップ](success-map.md#setup) → [商品を売る導線](success-map.md#sales)へ
* **商品もメールリストもある** → [ファネルでの販売](success-map.md#sales)や[高単価商品の個別相談](success-map.md#sales)、[会員サイト化](success-map.md#membership)で次の階段へ

## 共通セットアップ（すべての人の最初の一歩） <a href="#setup" id="setup"></a>

どのゴールを目指す場合も、まずこの4つを済ませておくと後がスムーズです。

1. [ウェブサイトとファネルについて](platform/getting-started/websites-and-funnels.md) — 2つの作り方の違いを理解する
2. [カスタムドメインの接続](platform/getting-started/custom-domain.md) — あなた独自のドメインで公開する
3. [メールドメインの接続](platform/email-and-automation/email-domain-connection.md) — メールがきちんと届くようにする
4. [ビルダーでページを編集する方法](platform/getting-started/edit-pages-in-builder.md) — ページ編集の基本操作を覚える

商品を販売する予定があれば、[決済サービスとの接続](platform/online-shop/payment-services.md)と[特商法の設定](platform/online-shop/tokushoho-setup-individual.md)もこの段階で済ませておきましょう。

## 🎯 メールリストを集めたい <a href="#leads" id="leads"></a>

見込み客の連絡先を集めて、こちらから届けられる関係を作ります。

1. [ファネルの種類一覧](funnel-designs/types-of-funnels/value-ladder-guide.md) — 全体像をつかむ
2. [リードジェネレーションファネル](funnel-designs/types-of-funnels/lead-generation.md)または[オプトインファネル](funnel-designs/types-of-funnels/opt-in.md) — 登録ページを作る
3. [デジタルダウンロードファネル](funnel-designs/types-of-funnels/digital-download-funnel.md) — 無料特典（リードマグネット）を配布する場合
4. [ダブルオプトイン確認メール](platform/system-emails/system-email-templates/double-opt-in-confirmation.md) — 登録確認メールを整える
5. [オートメーション概要](platform/email-and-automation/automation.md) — 登録後の自動メールを組む
6. [メールキャンペーン（メルマガ送信）](platform/email-and-automation/email-campaigns.md) — 集めたリストに配信する

## 💰 商品・サービスを売りたい <a href="#sales" id="sales"></a>

価格帯によって最適な売り方が変わります。まず[バリューラダー別ガイド](funnel-designs/types-of-funnels/value-ladder-guide.md)で全体像を見てから進むのがおすすめです。

**低〜中単価の商品（デジタル商品・講座・物販など）:**

1. [商品の作成](platform/online-shop/create-products.md) — 売るものを登録する
2. [トリップワイヤーファネル](funnel-designs/types-of-funnels/tripwire-funnel.md)や[VSLファネル](funnel-designs/types-of-funnels/vsl-funnel.md) — 販売ファネルを組む
3. [アップセル & ダウンセル](platform/funnels/upsell-downsell.md) — 1件あたりの売上を伸ばす
4. [A/Bスプリットテスト](platform/funnels/ab-split-testing.md) — 成約率を検証・改善する
5. [新規注文メール](platform/system-emails/system-email-templates/new-order.md) — 購入後の自動メールを整える

**高単価の商品（コーチング・コンサル・代行など）:**

1. [申込式ハイチケットファネル](funnel-designs/types-of-funnels/application-funnel.md) — 申込フォームと個別相談で売る型を理解する
2. [予約機能](platform/appointments/README.md) — 個別相談の日程調整を自動化する
3. [顧客管理（CRM）](platform/crm/README.md) — 申込者の情報とやり取りを一元管理する

## 🎓 会員サイト・オンライン講座を作りたい <a href="#membership" id="membership"></a>

購入者だけが見られるコンテンツを届けて、継続的な価値を提供します。

1. [オンラインコースファネル](funnel-designs/types-of-funnels/online-course-funnel.md)または[メンバーシップファネル](funnel-designs/types-of-funnels/membership-funnel.md) — 入口のファネルを選ぶ
2. [メンバーシップグループの作成](platform/crm/membership-groups.md) — 会員の区分を作る
3. [メンバーシップ限定コンテンツの設定](platform/crm/membership-content.md) — 会員だけが見られるコンテンツを設定する
4. [メンバー限定ページの作成](platform/website-funnel-builder/member-pages.md) — 会員専用ページを作る
5. [ビデオマネージャー](video/getting-started/video-manager.md)と[プレイリスト](video/getting-started/playlist.md) — 講座動画を配置する
6. [サブスクリプション管理](platform/online-shop/subscription-management.md) — 月額課金を運用する

## 👥 コミュニティを運営したい <a href="#community" id="community"></a>

メンバー同士が交流できる場を作り、エンゲージメントを高めます。

1. [コミュニティの作成](community/community-setup/README.md) — コミュニティの土台を作る
2. [サークル](community/circles/README.md) — テーマ別の交流スペースを設計する
3. [コミュニティウィジェット](community/community-widgets/README.md) — ディスカッションやウォール/フォーラムを設置する
4. [アクセスレベル](community/access-levels.md) — 誰がどこまで見られるかを設定する
5. [コミュニティ通知](community/community-notifications.md) — メンバーへの通知を整える

## 📣 ブログ・SNSで発信したい <a href="#content" id="content"></a>

コンテンツで見込み客との接点を増やします。AIを使えば、発信は思っているより軽くなります。

1. [ブログを始める](blog/starting-a-blog.md) — ブログの土台を作る
2. [AIプロフィール設定の使い方](blog/ai-profile-settings.md) — AIにあなたらしさを覚えさせる（最初の一度だけ）
3. [高品質記事生成AIの使い方](blog/premium-article-generator.md) — AIと一緒に記事を作る
4. [SNS運用AIの利用条件と申し込み方法](sns-unyou-ai/getting-started/eligibility-and-signup.md) — SNS投稿の自動生成を始める（Expandプラン以上）
5. [SNS投稿戦略ガイド](extra/sns-strategy-guide.md) — 発信の戦略を立てる

## 📅 予約を受け付けたい <a href="#booking" id="booking"></a>

面談・相談・レッスンなどの日程調整を自動化します。

1. [予約機能の概要](platform/appointments/README.md) — 2つの予約方式を理解する
2. [スケジュール予約の設定](platform/appointments/schedule-booking.md)または[サービス予約の設定](platform/appointments/service-based-booking.md) — 予約枠を作る
3. [アポイントメント予約ファネル](funnel-designs/types-of-funnels/appointment-booking.md) — 予約への導線ページを作る
4. [新規予約メール](platform/system-emails/system-email-templates/new-appointment.md)と[予約リマインダー](platform/system-emails/system-email-templates/appointment-reminder.md) — 自動通知を整える
